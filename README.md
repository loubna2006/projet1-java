# Application Java de Gestion d’un Tournoi de Football
## Description
Cette application Java permet de gérer un tournoi de football.
Elle facilite la gestion des équipes, des matchs et du classement.
##  Objectifs du projet
- Gérer les équipes
- Enregistrer les matchs et les scores
- Générer automatiquement le classement
- Réduire les erreurs humaines
##  Fonctionnalités
- Ajouter / modifier / supprimer une équipe
- Gérer les matchs
- Enregistrer les scores
- Afficher le classement
##  Technologies utilisées
- Java
- Java Swing
- JDK 8+
- IntelliJ IDEA / NetBeans
##  Architecture
- AppFoot : fenêtre principale
- TeamsPanel : gestion des équipes
- MatchesPanel : gestion des matchs
- TeamStat : calcul du classement
##  Installation
1. Installer JDK 8 ou plus
2. Cloner le projet
3. Ouvrir dans un IDE Java
4. Lancer la classe AppFoot
